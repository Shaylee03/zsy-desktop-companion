# 小样本原型验证

这份记录用于说明作品集页面中四个量化指标的口径。验证目标是检查固定触发、状态降级、个性化记忆三组策略在同一批原型样本上的相对差异。

## 样本口径

| 策略版本 | 说明 |
| --- | --- |
| 固定触发 | 按时间或事件直接介入，不充分考虑会议、专注和历史偏好。 |
| 状态降级 | 根据会议、专注、离席、返回等状态调整反馈强度。 |
| 个性化记忆 | 在状态降级基础上调用任务上下文、偏好、行为节奏和历史反馈。 |

每组使用 100 条原型回放样本，记录主动交互触发、会议 / 专注场景高打扰、用户标注反馈、离席返回任务恢复时间和记忆调用后纠正情况。

## 结果

| 指标 | 固定触发 | 个性化记忆 | 相对变化 |
| --- | ---: | ---: | ---: |
| 会议 / 专注场景高打扰触发率 | 41% | 15% | 下降 63% |
| 主动反馈接受率 | 42% | 55% | 提升 31% |
| 返回电脑后的任务恢复时间 | 125 秒 | 80 秒 | 缩短 36% |
| 记忆调用后用户纠正率 | 25% | 19% | 下降 24% |

## 计算依据

公开验证脚本：[`validation/small_sample_validation.py`](../validation/small_sample_validation.py)

后端同步脚本：`tools/small_sample_validation.py`

本地和云端测试项：`tests/test_small_sample_validation.py`

该脚本输出三组策略的样本计数，并由测试断言 README 与验证文档中的四个百分比，避免指标与验证口径脱节。

## 边界

- 这是小样本原型验证，适用于说明当前阶段的策略趋势。
- 指标用于说明策略方向和原型有效性，不代表长期留存或商业化效果。
- 样本中不包含服务器凭据、真实聊天正文、原始桌面日志或私人记忆内容。
