# 证据地图

这份文档用于说明项目页面中的主要说法分别由什么材料支撑，方便外部读者快速判断项目是否真实、完整、可运行。

## 快速查看路径

1. 首页：确认项目命题、核心场景和可下载体验。
2. [系统架构](architecture.md)：查看本地采集端、状态判断、策略层、反馈层和记忆层的职责边界。
3. [主动交互策略矩阵](interaction-strategy.md) 与 [决策流](decision-flow.md)：查看不同状态下如何选择 quiet / subtle / subtitle / speak。
4. [记忆进化机制](memory-evolution.md)：查看短期上下文、长期偏好、行为节奏和交互反馈如何影响策略。
5. [验证记录](validation.md)：查看当前原型完成的自动化检查、页面检查、下载包检查和云端烟测。
6. [实现匹配核查](implementation-audit-2026-06-26.md)：查看理想产品要求与当前实现的对应关系。
7. [下载说明](download-and-run.md)：查看如何启动本地采集端并连接自有后端和机器人。

## 说法与证据

| 页面说法 | 支撑材料 | 支撑强度 |
| --- | --- | --- |
| 项目关注主动介入边界 | 首页、产品案例、策略矩阵 | 高 |
| 使用非摄像头桌面信号 | 系统架构、隐私边界、本地启动包 | 高 |
| 具备四级反馈策略 | 首页、主动交互策略、决策流、策略测试 | 高 |
| 会议和专注中会降级 | 决策流、策略测试、云端后端模块 | 高 |
| 支持可下载体验 | Windows Starter Kit、下载说明、下载验证 | 高 |
| 设计分层记忆机制 | 记忆进化文档、架构文档 | 中高 |
| 支持离席返回和任务恢复 | 场景说明、后端策略钩子、策略测试 | 中高 |
| 已完成当前版本验证 | 验证记录、截图检查、远程服务烟测 | 高 |

## 已公开内容

- 产品定位和使用场景。
- 主动交互策略。
- 系统架构和字段口径。
- 记忆机制和生命周期。
- Windows 本地启动包。
- 原型验证记录。
- 隐私与实现边界。

## 不公开内容

- 服务器凭据、API Key、`.env`。
- 原始桌面日志、真实对话日志。
- 用户记忆、Hermes 会话、硬件固件。
- 任何个人后端 token 或本地 `config.local.json`。
