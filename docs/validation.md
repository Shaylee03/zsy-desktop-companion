# 验证记录

验证目标不是证明它已经是商业化产品，而是确认当前原型是否按“低打扰主动式桌面 AI 机器人”的核心规则工作：

- 会议中不应突然语音打断。
- 深度专注中应延后普通提醒。
- 离席返回应优先字幕摘要。
- 任务恢复应提供低压力提示。
- 下载包应可获取、可配置、可停止。
- 页面应能让外部读者快速理解项目。
